## ✨ Funcionalidades

| Módulo | Descripción |
|--------|-------------|
| 📊 **Dashboard** | Métricas clave, gráfica de ingresos, progreso del pipeline y actividades próximas |
| 👥 **Contactos** | CRUD completo con búsqueda, filtros por estado y tags personalizados |
| 💼 **Tratos** | Pipeline Kanban (Discovery → Ganado/Perdido) + vista Lista con probabilidad de cierre |
| 📅 **Actividades** | Seguimiento de llamadas, emails y reuniones con checklist de completado |
| 📈 **Reportes** | Análisis de ingresos, distribución del pipeline y tasa de cierre con gráficas interactivas |
| ⚙️ **Configuración** | Perfil de usuario, notificaciones, seguridad (2FA) e información del plan |

## 🎨 Guía de diseño

| Token | Valor | Uso |
|-------|-------|-----|
| `ink-950` | `#0a0a0f` | Fondo base |
| `ink-900` | `#15151e` | Tarjetas |
| `ink-800` | `#2a2a37` | Inputs y bordes hover |
| `acid` | `#c8ff00` | Acento principal / CTA |
| `coral` | `#ff4f5e` | Alertas y eliminación |
| `sky` | `#00c8ff` | Estado "frío" / info |

**Tipografía:**  `Syne` (títulos) · `DM Sans` (cuerpo) · `JetBrains Mono` (datos y badges)
